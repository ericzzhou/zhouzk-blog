# Prompts 收藏库

本目录用于收集和整理日常工作中实用的 Prompt 模板。

---

```

    ├── 周报生成.md            # 工作周报整理
    └── 会议纪要整理.md        # 会议记录结构化
```

## 使用建议

1. **直接复制**：每个 `.md` 文件都可以直接复制内容使用
2. **按需修改**：根据实际项目情况调整方括号 `[xxx]` 中的变量
3. **持续积累**：发现好用的 Prompt 及时添加，保持更新

---

## 添加新 Prompt 的规范

1. **文件命名**：使用简短中文描述，如 `代码审查.md`
2. **分类存放**：根据主要用途放入对应文件夹
3. **头部信息**：每个文件开头建议包含：
   - 用途说明
   - 适用场景
   - 变量说明（如有 `[xxx]` 等需替换的内容）
4. **更新索引**：在 README.md 的表格中添加新条目

---

## 变量约定

本库中的 Prompt 使用以下占位符约定：

| 占位符 | 含义 | 示例 |
|--------|------|------|
| `[xxx]` | 项目名称/核心主题 | `[电商后台]`、`[用户增长策略]` |
| `[技术栈]` | 具体技术选型 | `React + TypeScript` |
| `[输出格式]` | 期望的输出形式 | `Markdown 表格`、`JSON` |
