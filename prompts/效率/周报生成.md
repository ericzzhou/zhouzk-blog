# 周报生成 Prompt

## 用途
根据零散的工作记录，生成结构清晰、重点突出的周报。

## 适用场景
- 每周五快速整理周报
- 向领导汇报工作进展
- 记录个人工作轨迹

---

## Prompt 内容

```
请根据我提供的工作记录，生成一份结构清晰的周报。

我的工作记录：
[粘贴你的零散记录，如：
- 周一：修复了登录页面的 Bug
- 周二：参加了需求评审会
- 周三：开发了用户管理模块的列表页
...]

请按以下格式输出：

## 本周工作总结

### 重点项目进展
- [项目名]
  - 进展：[具体完成情况]
  - 卡点：[如有，说明原因和需要的支持]
  - 下周计划：[下一步做什么]

### 日常事务
- [列出其他工作事项]

### 学习与成长
- [本周学到的新东西]

---

## 下周工作计划
1. [优先级最高的事]
2. [次要事项]
3. [如果有余力想做的事]

---

要求：
1. 用"完成"而非"做了"，强调结果
2. 适当归类合并相似事项
3. 突出关键成果，不堆砌细节
4. 语气专业但不过分正式
```

---

## 进阶用法

如果有 OKR 或季度目标，可以在开头加上：

```
我的季度目标是：[目标描述]
请确保周报内容与此对齐，并说明每项工作对目标的贡献。
```
