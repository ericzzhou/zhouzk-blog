# 小红书文案生成 Prompt

## 用途
生成符合小红书平台调性的种草文案，包括标题、正文、标签。

## 适用场景
- 产品推广
- 个人经验分享
- 内容营销

---

## Prompt 内容

```
请帮我写一篇小红书风格的种草文案。

产品/主题：[描述你要推广的内容]
核心卖点：[最重要的 2-3 个亮点]
目标人群：[面向谁，如：25-30 岁职场女性]
使用场景：[在什么情况下用，如：通勤、约会、居家]
风格偏好：[可爱/专业/极简/高级/接地气]
字数要求：[建议 300-600 字]

请输出：
1. **标题**（3个选项）
   - 要求：有emoji、有数字或具体效果、制造好奇或共鸣

2. **正文**
   - 开头：痛点共鸣或场景代入（2-3句）
   - 中间：产品介绍 + 真实使用体验 + 效果对比
   - 结尾：引导互动（提问或号召收藏/点赞）
   - 语言：口语化、多用短句、适当用emoji分隔

3. **标签**（8-10个）
   - 包含大标签（流量高）和精准标签（转化高）

4. **配图建议**
   - 描述每张照片应该展示什么角度/场景

参考调性：
- 真实感：像朋友推荐，不是广告
- 价值感：读者看完觉得有用/有趣/有共鸣
- 互动感：引导评论和收藏
```

---

## 使用技巧

如果想更真实，可以加上：
```
请模仿一个[身份，如：互联网打工人/新手妈妈]的真实分享口吻，
不要过度营销感，像日常笔记一样自然。
```
